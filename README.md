# BYOND-2.0

BYOND-2.0 — это игровой движок с открытым исходным кодом, вдохновленный [BYOND](https://www.byond.com/). Он разработан для того, чтобы быть легким, гибким и простым в использовании.

## Функции

*   **Lua-скрипты**: Используйте Lua для написания игровой логики.
*   **Горячая перезагрузка**: Автоматическая перезагрузка скриптов при их изменении.
*   **Кроссплатформенность**: Работает на Windows, macOS и Linux.

## Начало работы

### Требования

*   [.NET 8.0 SDK](httpshttps://dotnet.microsoft.com/download/dotnet/8.0)

### Сборка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/jojorwu/BYOND-2.0.git
    cd BYOND-2.0
    ```

2.  **Соберите проект:**
    ```bash
    dotnet build
    ```

3.  **Запустите сервер:**
    ```bash
    dotnet run --project Server/Server.csproj
    ```
