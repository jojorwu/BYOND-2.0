# Руководство по редактору

Редактор BYOND 2.0 — это графическое приложение, созданное с помощью Silk.NET и ImGui, которое позволяет разработчикам создавать и изменять игровые карты (файлы `.dmm`) и определять свойства объектов.

*(Примечание: В будущем это руководство будет дополнено скриншотами для лучшей иллюстрации интерфейса.)*

## Как запустить редактор

Самый простой способ запустить редактор — использовать предоставленный shell-скрипт из корня проекта:

```bash
./run_editor.sh
```

## Обзор пользовательского интерфейса

Интерфейс редактора состоит из нескольких ключевых панелей:

*   **Окно карты (Map View):** Центральная панель, которая предоставляет визуальное, основанное на тайлах, представление вашей игровой карты. Здесь вы можете перемещаться по карте, выбирать тайлы и размещать объекты.
*   **Дерево объектов (Object Tree):** Эта панель, расположенная слева, отображает иерархическое дерево всех доступных типов объектов, определенных в вашем DM-коде (например, `/obj`, `/turf`, `/mob`). Она используется для выбора объектов для размещения.
*   **Панель свойств (Properties Panel):** Эта панель, расположенная справа, отображает свойства текущего выбранного объекта или тайла на карте. Вы можете просматривать и редактировать такие свойства, как переменные объекта или тип тайла.
*   **Строка меню (Menu Bar):** В верхней части окна строка меню предоставляет доступ к файловым операциям (Создать, Открыть, Сохранить) и другим функциям редактора.

## Основной рабочий процесс

### Создание новой карты

1.  Перейдите в `Файл > Новая карта` в строке меню.
2.  Появится диалоговое окно с запросом размеров карты (ширина и высота в тайлах).
3.  Введите желаемые размеры и нажмите "Создать". В окне карты будет создана новая пустая карта.

### Размещение объектов

1.  **Выберите тип объекта:** На панели **Дерево объектов** перейдите по иерархии и щелкните на тип объекта, который вы хотите разместить (например, `/obj/item/apple`).
2.  **Выберите тайл:** В **Окне карты** щелкните левой кнопкой мыши на тайл, где вы хотите разместить объект.
3.  Экземпляр выбранного объекта будет размещен на этом тайле. Вы можете разместить несколько объектов, многократно щелкая по разным тайлам.

### Редактирование свойств

1.  **Выберите экземпляр:** В **Окне карты** щелкните правой кнопкой мыши на тайл, содержащий объект, который вы хотите проверить.
2.  **Просмотрите свойства:** **Панель свойств** справа обновится, чтобы показать все переменные и их текущие значения для выбранного экземпляра объекта.
3.  **Измените свойства:** Вы можете щелкнуть по значениям в панели свойств, чтобы отредактировать их напрямую. Например, вы можете изменить переменную `bites_left` у яблока еще до начала игры.

### Сохранение карты

1.  Перейдите в `Файл > Сохранить карту как...` в строке меню.
2.  Откроется диалоговое окно для сохранения файла. Перейдите в каталог `maps`, введите имя для вашей карты (например, `main.dmm`) и нажмите "Сохранить".

Теперь ваша карта сохранена и может быть загружена сервером во время выполнения.
