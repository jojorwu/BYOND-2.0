# Как внести свой вклад

Мы рады, что вы хотите внести свой вклад в BYOND-2.0! Любая помощь приветствуется.

## Сообщения об ошибках

Если вы нашли ошибку, пожалуйста, создайте [issue](https://github.com/jojorwu/BYOND-2.0/issues) и предоставьте следующую информацию:

*   **Описание ошибки**: Подробное описание того, что произошло.
*   **Шаги для воспроизведения**: Что нужно сделать, чтобы ошибка повторилась.
*   **Ожидаемое поведение**: Что должно было произойти.
*   **Скриншоты**: Если возможно, приложите скриншоты.

## Pull Request'ы

1.  Сделайте форк репозитория.
2.  Создайте новую ветку (`git checkout -b my-feature-branch`).
3.  Внесите свои изменения.
4.  Закоммитьте свои изменения (`git commit -am 'Add some feature'`).
5.  Отправьте в ветку (`git push origin my-feature-branch`).
6.  Создайте новый Pull Request.
