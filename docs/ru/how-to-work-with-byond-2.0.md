# Как работать с BYOND 2.0

Это руководство поможет вам настроить рабочее окружение, собрать проект и запустить сервер BYOND 2.0.

## 1. Настройка окружения

Для работы с проектом вам понадобится .NET 10.0 SDK.

### Установка .NET 10.0 SDK

Вы можете установить .NET 10.0 SDK, выполнив следующие команды в корневой папке проекта:

```bash
chmod +x ./dotnet-install.sh
./dotnet-install.sh --channel 10.0
```

После установки необходимо добавить .NET в `PATH` для текущей сессии:

```bash
export DOTNET_ROOT=$HOME/.dotnet
export PATH=$PATH:$DOTNET_ROOT:$DOTNET_ROOT/tools
```

## 2. Сборка проекта

После установки .NET SDK вы можете собрать проект. Рекомендуется собирать решение целиком, но можно и отдельные проекты. Для сборки всего решения выполните команду:

```bash
dotnet build BYOND2.0.sln
```

## 3. Запуск сервера

Для запуска игрового сервера выполните следующую команду:

```bash
dotnet run --project Server/Server.csproj
```

После запуска сервер начнет отслеживать изменения в скриптах в каталоге `scripts` и будет автоматически перезагружать их при необходимости.
