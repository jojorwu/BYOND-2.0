# Editor Guide

The BYOND 2.0 Editor is a graphical application that allows developers to create and modify game maps and other assets.

## Key Features

*   **Map Creation:** The editor provides a visual interface for creating and editing game maps (`.dmm` files).
*   **Object Editing:** It allows for the placement and configuration of game objects on the map.
*   **Asset Management:** The editor can be used to manage other game assets, such as icons and sounds.

## How to Run the Editor

To run the editor, execute the following command from the root of the project:

```bash
dotnet run --project Editor/Editor.csproj
```

The editor will launch, and you can start creating or editing your game assets.
