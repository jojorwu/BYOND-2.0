# Руководство по редактору

Редактор BYOND 2.0 — это графическое приложение, которое позволяет разработчикам создавать и изменять игровые карты и другие ресурсы.

## Ключевые особенности

*   **Создание карт:** Редактор предоставляет визуальный интерфейс для создания и редактирования игровых карт (файлов `.dmm`).
*   **Редактирование объектов:** Он позволяет размещать и настраивать игровые объекты на карте.
*   **Управление ресурсами:** Редактор можно использовать для управления другими игровыми ресурсами, такими как значки и звуки.

## Как запустить редактор

Чтобы запустить редактор, выполните следующую команду из корня проекта:

```bash
dotnet run --project Editor/Editor.csproj
```

Редактор запустится, и вы сможете приступить к созданию или редактированию своих игровых ресурсов.
