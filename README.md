# BYOND-2.0

BYOND-2.0 - это кроссплатформенный игровой движок с открытым исходным кодом, вдохновленный BYOND. Он построен на .NET с использованием клиент-серверной архитектуры.

## Обзор архитектуры

Проект состоит из нескольких ключевых компонентов:

-   **Server:** Основное серверное приложение. Оно управляет игровым миром, обрабатывает сетевые подключения и исполняет игровой код.
-   **Core:** Общая библиотека, содержащая основную логику, используемую всеми компонентами. Сюда входят модели данных, базовые сервисы и, что самое важное, виртуальная машина (ВМ).
-   **VM (внутри Core):** Стековая виртуальная машина, которая исполняет скомпилированный байт-код. Она поддерживает вызовы процедур, аргументы, локальные переменные и основные операции.
-   **Compiler:** Компилятор, который преобразует исходный код DM (DREAM Maker) в байт-код для исполнения виртуальной машиной.
-   **Client:** Графический клиент, который подключается к серверу для рендеринга игрового мира и обработки ввода пользователя.
-   **Editor:** Графический редактор для создания и редактирования игровых карт (`.dmm` файлов).

Более подробную информацию об архитектуре можно найти в [docs/architecture.md](docs/architecture.md).
