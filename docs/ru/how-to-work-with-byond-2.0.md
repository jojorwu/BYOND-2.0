# Как работать с BYOND 2.0

Это руководство поможет вам настроить рабочее окружение, собрать проект и запустить сервер BYOND 2.0.

## 1. Настройка окружения

Для работы с проектом вам понадобится .NET 8.0 SDK.

### Установка .NET 8.0 SDK

В репозитории есть скрипт для установки правильной версии .NET SDK. Выполните следующие команды из корня проекта:

```bash
chmod +x ./dotnet-install.sh
./dotnet-install.sh --channel 8.0
```

После установки необходимо добавить .NET в `PATH` для текущей сессии:

```bash
export DOTNET_ROOT=$HOME/.dotnet
export PATH=$PATH:$DOTNET_ROOT:$DOTNET_ROOT/tools
```

## 2. Сборка проекта

После установки .NET SDK вы можете собрать проект. Для сборки всего решения выполните следующую команду из корня проекта:

```bash
dotnet build BYOND2.0.sln
```

## 3. Запуск сервера

Самый простой способ запустить сервер — использовать предоставленный shell-скрипт:

```bash
./run_server.sh
```

Этот скрипт соберет необходимые проекты и запустит сервер. После запуска сервер начнет отслеживать каталог `scripts` и автоматически будет перезагружать любые изменения, которые вы вносите в файлы скриптов.
