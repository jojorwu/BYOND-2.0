# Руководство: Создание объекта

Это руководство проведет вас через процесс создания нового типа игрового объекта с использованием DM-скрипта, а затем создания его экземпляра.

## 1. Определение типа объекта

Сначала нам нужно определить новый тип объекта. Давайте создадим простой объект "Яблоко", который можно съесть. Создайте новый файл с именем `scripts/apple.dm` и добавьте следующий код:

```dm
// scripts/apple.dm
/obj/item/apple
    var/bites_left = 3

    verb/eat()
        set src in usr

        if (bites_left > 0)
            bites_left--
            usr << "Вы откусываете яблоко. Осталось [bites_left] укусов."
            if (bites_left == 0)
                usr << "Вы доели яблоко."
                del src
        else
            usr << "Больше нечего есть!"
```

Этот код определяет новый тип объекта `/obj/item/apple` с переменной `bites_left` и глаголом `eat()`.

## 2. Создание экземпляра объекта

Теперь давайте создадим экземпляр нашего нового объекта-яблока в игровом мире. Мы можем сделать это в нашей основной точке входа DM. Если у вас нет файла `main.dm`, создайте его в каталоге `scripts`.

```dm
// scripts/main.dm
/world/New()
    // Создать яблоко в координатах (5, 5)
    new /obj/item/apple(locate(5, 5, 1))
```

## 3. Запуск сервера

Теперь запустите сервер:

```bash
dotnet run --project Server/Server.csproj
```

Сервер скомпилирует ваши DM-скрипты и создаст яблоко в игровом мире. Затем вы сможете подключиться с помощью клиента и взаимодействовать с яблоком, используя глагол "eat".
