# Руководство: Создание объекта

Это руководство проведет вас через процесс создания нового типа игрового объекта с использованием DM-скрипта, а затем создания его экземпляра.

## 1. Определение типа объекта

Сначала нам нужно определить новый тип объекта. Давайте создадим простой объект "Яблоко", который можно съесть. Создайте новый файл с именем `scripts/apple.dm` и добавьте следующий код:

```dm
// scripts/apple.dm
/obj/item/apple
    var/bites_left = 3 // Сколько укусов осталось

    // Глагол 'eat' позволяет игрокам взаимодействовать с яблоком.
    verb/eat()
        set src in usr // Этот глагол нацелен на пользователя (игрока).

        if (bites_left > 0)
            bites_left--
            usr << "Вы откусываете яблоко. Осталось [bites_left] укусов."
            if (bites_left == 0)
                usr << "Вы доели яблоко."
                del src // Удаляем объект, когда он съеден.
        else
            usr << "Больше нечего есть!"
```

Этот код определяет новый тип объекта `/obj/item/apple`, который наследуется от `/obj/item`. У него есть переменная `bites_left` и глагол `eat()`.

## 2. Создание экземпляра объекта

Теперь давайте создадим экземпляр нашего нового объекта-яблока в игровом мире. Мы можем сделать это в основной процедуре мира `New()`, которая вызывается при запуске сервера. Если у вас нет файла `main.dm`, создайте его в каталоге `scripts`.

```dm
// scripts/main.dm
/world/New()
    // Создаем яблоко в координатах X=5, Y=5 на Z-уровне 1.
    // Процедура `locate()` находит turf (тайл) по указанным координатам.
    // `new /obj/item/apple(...)` создает новое яблоко в этой локации.
    new /obj/item/apple(locate(5, 5, 1))
```

## 3. Запуск сервера

Теперь запустите сервер из корня проекта:

```bash
./run_server.sh
```

Сервер скомпилирует ваши DM-скрипты и создаст яблоко в игровом мире. Затем вы сможете подключиться с помощью клиента и взаимодействовать с яблоком, используя глагол "eat".
