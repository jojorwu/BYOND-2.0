# Руководство по клиенту

Клиент BYOND 2.0 — это консольное приложение, отвечающее за отрисовку игрового мира, обработку пользовательского ввода и взаимодействие с игровым сервером.

## Ключевые обязанности

*   **Отрисовка:** Клиент отрисовывает состояние игры, полученное от сервера.
*   **Пользовательский ввод:** Он захватывает ввод с клавиатуры и мыши и отправляет его на сервер.
*   **Сеть:** Клиент устанавливает и поддерживает соединение с сервером, получая обновления состояния игры и отправляя команды пользователя.

## Как запустить клиент

Чтобы запустить игровой клиент, сначала убедитесь, что сервер запущен. Затем выполните следующую команду из корня проекта:

```bash
dotnet run --project Client/Client.csproj
```

После запуска клиент попытается подключиться к серверу, указанному в его конфигурации.
